## ğŸ’» Cara Menjalankan (Instalasi)

Pastikan Anda telah menginstal **Node.js** di komputer Anda.

1.  **Clone repositori ini** (atau unduh zipnya):
    ```bash
    git clone https://github.com/JustRio24/sweetmelt-cookies-website.git
    cd sweetmelt-cookies-website/projek-sintak
    ```
    *(Sesuaikan path direktori jika perlu)*

2.  **Instal dependensi**:
    ```bash
    npm install
    ```

3.  **Jalankan server development**:
    ```bash
    npm run dev
    ```

4.  **Buka di browser**:
    Kunjungi tautan yang muncul di terminal (biasanya `http://localhost:5173`).
